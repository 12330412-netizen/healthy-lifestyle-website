# Healthy LifeStyle Website
# Project description 
The healthy lifestyle website is a responsive web application that promotes healthy living through fitness,nutrition,and wellness information.
# Technologies used:
-react.js
-CSS
# setup instructions 
1-clone the repository 
2-run npm install
3- run npm start
